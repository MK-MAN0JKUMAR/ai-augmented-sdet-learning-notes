# AI-Augmented SDET Learning

A comprehensive, self-paced learning roadmap for test automation engineers (SDETs) to master AI-integrated testing and advance their careers.

---

## What Is This?

This repository contains for SDETs to transition from traditional test automation to AI-augmented systems. It covers:

- Python and LLM fundamentals
- Evaluation frameworks (DeepEval, RAGAS)
- RAG systems and evaluation
- AI integration in automation
- Job search and interview preparation

**All self-paced. All hands-on. All practical.**

---

## Why This Repository?

The market is shifting. Traditional SDET skills are becoming commoditized.

But SDETs who can:
- Test non-deterministic AI systems
- Design evaluation frameworks for LLMs
- Build RAG-based automation
- Integrate AI into existing frameworks

This repo teaches exactly that.

---

## The 4 Phases

### Phase 1: Python & LLM Basics
Learn Python, LLM concepts, Groq API, Ollama. Build your first evaluator.

**Exit:** Working evaluator script  
**See:** `phase-1-python-llm-basics/`

---

### Phase 2: LLM Evaluation & Testing
Master DeepEval and RAGAS metrics. Build automated test suites for AI.

**Exit:** Full test suite with evaluation reports  
**See:** `phase-2-llm-evaluation/`

---

### Phase 3: RAG Testing Framework
Build a production RAG system with evaluation layer. Create portfolio piece.

**Exit:** GitHub portfolio project  
**See:** `phase-3-rag-framework/`

---

### Phase 4: Playwright + AI Integration
Add AI to your existing automation framework. Show measurable impact.

**Exit:** Enhanced framework with AI layer  
**See:** `phase-4-playwright-integration/`

## FAQ

**Q: I'm busy. Can I do this faster?**  
A: This is self-paced. Do it in 12 months if you commit 10+ hours/week. But don't skip phases.

**Q: What if I get stuck?**  
A: Check the study guide's troubleshooting section. Ask in GitHub Issues. Read the resources.

**Q: Can I do phases in parallel?**  
A: No. Each phase builds on previous knowledge. Complete in order.

**Q: Do I need all the tools?**  
A: Phase 1–2 need Python + Groq/Ollama. Phase 3 adds FAISS. Phase 4 adds Playwright. Phase 5 is interviews.

**Q: Will this get me a job?**  
A: If you complete all phases and build projects, yes. The roadmap is designed for job market fit.

**Q: What if I already know some of this?**  
A: Skim that phase, focus on gaps. But build the exit projects anyway—they matter for portfolio.
