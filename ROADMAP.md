## Phase 1: Python & LLM Basics
**What you learn:**
- Python fundamentals (variables, functions, loops, file handling)
- LLM concepts (tokens, temperature, prompts)
- Groq API and Ollama local models
- Non-deterministic system testing basics
- Build: Basic LLM evaluator script

**Exit:** Can call LLM APIs and write basic evaluator

---

## Phase 2: LLM Evaluation & Testing
**What you learn:**
- DeepEval framework (hallucination, relevance metrics)
- RAGAS framework (faithfulness, context recall/precision)
- Evaluation strategies (offline vs online)
- Cost vs accuracy trade-offs
- Hybrid evaluation approach
- Build: Full test suite with evaluation metrics

**Exit:** Can build automated test suites for AI outputs

---

## Phase 3: RAG Testing Framework
**What you learn:**
- RAG pipeline (retrieval + generation)
- FAISS vector database
- End-to-end RAG evaluation
- Portfolio project with GitHub
- Build: Complete RAG system with test layer

**Exit:** Portfolio piece showing RAG + evaluation

---

## Phase 4: Playwright + AI Integration
**What you learn:**
- AI-powered assertions (semantic validation)
- Failure analysis using LLMs
- Auto test generation from requirements
- Measuring impact (reduce flaky tests)
- Build: Existing Playwright framework + AI layer

**Exit:** Framework with visible AI features

---

## Success Markers

**Phase 1:** Working evaluator script  
**Phase 2:** Full test suite with HTML/JSON reports  
**Phase 3:** Portfolio project on GitHub  
**Phase 4:** Automation framework with AI layer
